# 切换到黑暗模式

1. 升级到 78+

2. 地址栏输入 `chrome://flags/#enable-force-dark`

3. `Force Dark Mode for Web Contents` 右边选择 __Enabled__ (默认为 __Default__)

4. 点击下方弹出的的 Relaunch 重启 Chrome

# 切换到正常模式

1. 升级到 78+

2. 地址栏输入 `chrome://flags/#enable-force-dark`

3. `Force Dark Mode for Web Contents` 右边选择 __Default__

4. 点击下方弹出的的 Relaunch 重启 Chrome

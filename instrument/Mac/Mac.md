
# 快捷键

| 描述 | 快捷键 |
| :--- | :--- |
| 整屏截图（保存到图片） | command + shift + 3 |
| 整屏截图（保存到剪切板） | control + command + shift + 3 |
| 部分截图（保存到图片） | command + shift + 4 |
| 部分截图（保存到剪切板） | control + command + shift + 4 |

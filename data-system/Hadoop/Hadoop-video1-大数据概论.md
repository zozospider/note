# 大数据部门业务流程分析

1. 产品人员提需求 (统计总用户数, 日活跃用户数, 回流用户数)
2. 数据部门搭建数据平台, 分析数据指标
3. 数据可视化 (报表展示, 邮件发送, 大屏展示等)

# 大数据部门组织结构

- 平台组
  - Hadoop, Flume, Kafka, HBase, Spark 等框架平台搭建
  - 集群性能监控
  - 集群性能调优
- 数据仓库组
  - ETL 工程师: 数据清洗
  - Hive 工程师: 数据分析, 数据仓库建模
- 数据挖掘组
  - 算法工程师
  - 推荐系统工程师
  - 用户画像工程师
- 报表开发组
  - JavaEE 工程师
